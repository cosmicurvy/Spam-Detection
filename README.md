# Spam Detection

This project uses Natural Language Processing (NLP) and machine learning to classify SMS messages as either spam or ham (not spam). The application preprocesses text, extracts numerical features using CountVectorizer, and predicts a message category through a trained machine learning model deployed with Flask.
